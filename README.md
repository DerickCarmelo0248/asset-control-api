Asset Control API

API REST desenvolvida para controle de ativos de TI em ambiente corporativo.

O projeto simula a necessidade real de controle de equipamentos de informática, incluindo registro de ativos, movimentações e status.

🚀 Tecnologias utilizadas

Node.js

Express

PostgreSQL

pg (driver nativo)

dotenv

Arquitetura modular

🎯 Objetivo

Desenvolver uma API backend sólida, aplicando boas práticas de arquitetura, modelagem de banco de dados e organização de código.

O projeto faz parte da minha transição estratégica da área de Infraestrutura para Desenvolvimento Backend.

📌 Funcionalidades previstas

Cadastro de equipamentos

Controle de status (disponível, em uso, manutenção)

Registro de movimentações

Histórico de alterações

Controle de usuários (futuro)

📂 Estrutura do Projeto
src/
 ├── controllers/
 ├── routes/
 ├── database/
 ├── config/
 └── server.js
🔧 Como executar

Instalar dependências:

npm install

Configurar variáveis de ambiente (.env)

Rodar:

npm run dev
